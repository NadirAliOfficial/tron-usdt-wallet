# Tron USDT Wallet

A Python utility to create Tron wallets and test USDT transactions on the Tron network.

## Features
- Generate new Tron wallets (address + private key)
- Check USDT (TRC-20) balance
- Send test transactions

## Requirements
```
pip install tronpy
```

## Usage
```bash
python wallet.py --generate
python wallet.py --balance <address>
```
